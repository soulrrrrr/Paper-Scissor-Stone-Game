# Paper Scissor Stone Game

Think carefully. Beat your opponent.

## How to use

You can click the link in the description above or deploy server by yourself:

1. Clone this repository

2. Open the folder, go to /server

3. Run
```
npm install
node app.js
```

4. Go back to /, run
```
npm install
npm start
```
